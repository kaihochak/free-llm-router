# Privacy Policy for Free LLM Router

**Last Updated:** January 2026

## 1. Information We Collect

### Account Information

- Email address
- GitHub username and profile (if using GitHub OAuth)
- Account creation date

### API Usage Data

- API key names and metadata
- Endpoint usage (which endpoints you call)
- Request timestamps
- Response times
- HTTP status codes
- Approximate request volume

### Health Feedback Data

- Model IDs you report on
- Success/failure indicators
- Issue types (rate limiting, unavailable, error)
- Optional error details you provide
- Timestamp of report

## 2. How We Use Your Information

### Service Operation

- Authenticate your API requests
- Track rate limit usage
- Monitor system performance
- Prevent abuse and fraud

### Improvement

- Analyze usage patterns to optimize the API
- Understand which models are most used
- Improve health data quality
- Fix bugs based on error reports

### Communication

- Send notifications about service status
- Respond to support requests
- Send important security announcements

## 3. Data Sharing

### Public Data

- Aggregated health metrics (error rates, availability) are public
- Model availability and metadata are public

### Private Data

- Your personal health reports are never shared publicly
- Your API usage logs are private to you
- Your account information is not shared with third parties

### Third Parties

We do not sell your data. We may share data with:

- **Service Providers:** Database hosting (Neon), hosting provider (Cloudflare)
- **Legal Requirements:** If required by law or court order

## 4. Data Retention

| Data Type        | Retention Period       | Purpose                                    |
| ---------------- | ---------------------- | ------------------------------------------ |
| Request Logs     | Indefinite (currently) | Performance monitoring and abuse detection |
| Health Feedback  | Indefinite             | Historical model reliability data          |
| API Key Activity | Indefinite             | Security and usage tracking                |
| Account Data     | Until deletion         | Service access and authentication          |

**Note:** We plan to implement automatic cleanup of request logs older than 30 days in the future. Currently, all data is retained indefinitely unless you request deletion.

## 5. Your Rights

### Access

- You can view your API keys, usage statistics, and submitted feedback anytime

### Correction

- Update your email address and account details
- Correct health reports you've submitted

### Deletion

- Delete individual API keys
- Request full account deletion (will remove all associated data)

### Data Export

- Request a copy of your data (email us or create a GitHub issue)

## 6. Security

### Encryption

- API keys are hashed with SHA-256 before storage
- HTTPS required for all API requests
- Database credentials stored securely

### Best Practices

- We regularly update dependencies
- Perform security audits
- Never log sensitive information (API key values)

## 7. Children

Our service is not intended for users under 13. We do not knowingly collect information from children.

## 8. Self-Hosted Instances

This privacy policy applies only to the hosted instance at api.freemodels.ai. If you self-host, you are responsible for your own privacy practices.

## 9. International Data

Our primary servers are in the US. If you're in the EU or other jurisdictions with data protection laws, please note that your data may be processed in the US.

## 10. Changes to Policy

We may update this policy. We will notify you of significant changes via email or by updating the "Last Updated" date.
